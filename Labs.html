LABS

Lab filtering

Press the button bellow to get all labs.

If you want to get a specific lab and its chemicals and equipment, type bellow the ID and press "Submit" button.


Bellow you can get the labs by their location. The procedure is almost identical to the ID filtering, simply type in the location and click "Submit".


Post a new lab

If you want to introduce a new lab, fill in the form bellow and press "Post button". The ID will be generated automatically. The created lab will have empty lists of chemicals and equipment, so be sure to add some too!

Add a chemical to the lab

Be sure to choose the lab in ID filtering!

Fill in the form. Again, ID will be generated automatically then press the "Post" button.

Add a piece of equipment to the lab

Be sure to choose the lab in ID filtering!

Same as for the chemical - fill the form, press the "Post" button.

Update lab

Be sure to choose the lab in ID filtering and type in the ID again! This way you make sure that you want to update this exact lab.

Fill in the form, don't worry, the chemicals and equipment lists will not be emptied! Be sure to press the "Put" button.

Delete a lab

Be sure to choose the lab in ID filtering!

To delete a selected lab, simply press the button down bellow. Easy!
